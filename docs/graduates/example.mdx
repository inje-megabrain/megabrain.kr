# 성함

## 작성
현재의 url 경로는 `/docs/graduates/파일명`입니다. 이 문서는 `markdownX`라는 마크업 언어로 작성되어 있습니다. 마크다운 문법과 js 기능을 일부 이용할 수 있습니다. 글 작성에 도움이 될 레퍼런스를 첨부합니다.
- [mdx 문서 생성](https://docusaurus.io/docs/create-doc)
- [React를 활용한 mdx 기능 활용](https://docusaurus.io/docs/markdown-features/react)

## 기수 입력
기수 입력은 필수입니다. 기수는 mdx의 메타데이터 중 `siderbar_position`에 입력해주세요. 19년도에 메가브레인에 들어왔다면 `sidebar_position: 19`로 입력해주세요.
```
---
sidebar_position: 19
---
# 예제 제목

## 작성
...
```

## 이미지 업로드
이미지 첨부는 현재 폴더에서 `./static`디렉터리에 업로드합니다. 다음과 같이 입력하시면됩니다.

```markdown
![](./static/megabrain-logo.png)
```

![](./static/megabrain-logo.png)

README.md을 읽어보시고 실행하고 확인해보시면서 작업해보시는걸 추천드립니다.
